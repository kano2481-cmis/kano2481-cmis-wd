Procedure to Create and View a Website on the Production Server

  1. Open the html and css files needed on gedit
  2. Once the editing of the website is done, Validate the source code.
  3. Make sure that there are no errors in the source code.
  4. In terminal type in, sftp kano2481@cmis-hsteacher
  5. Type in password for it (longer one in caps)
  6. Type in, cd public_html
  7. Type in, pwd to view which folder you are in. Make sure that you are in public_html
  8. Type in, "put" and the name of the folder you are wanting to publicate
  9. Check on cmis-hsteacher/~kano2481/filename to make sure that the page is publicated.
